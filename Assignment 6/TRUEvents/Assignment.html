Displaying the Daily Events at TRU Student Union

The TRU Student Union is a gathering place for students and faculty at Thompson Rivers University. A Web site has been created in order to better advertise the union and its activities. You've been asked to work on the Web page that will list the daily union events.

Union events are usually printed out and displayed on bulletin boards and kiosks around campus. In order to keep the daily events current, this daily event sheet will be scanned and saved as TRU, which can then be inserted into the Web page each day. There is also a TRU Student Union "mascot" who regularly appears on the event sheet. Normally, there is a different picture of the mascot for each day of the week.

The page you'll create will use JavaScript to display the current day and date, as well as a different logo for each day of the week. In fact, the entire contents of the Web page should be generated by JavaScript.
